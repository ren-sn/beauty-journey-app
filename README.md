# 🌸 21天变美打卡

一个温馨可爱的打卡应用，帮助你和姐妹们一起变美～

## ✨ 功能特点

- 📱 **移动端友好**：适合手机使用，界面温馨可爱
- 📅 **21天打卡**：经典的21天习惯养成计划
- 📸 **图片分享**：支持上传照片，记录美好时刻
- 📊 **进度追踪**：可视化的打卡日历和进度
- 👥 **多人使用**：支持多用户，可在微信群分享
- 🔐 **管理员后台**：查看用户数据，下载统计
- 🎨 **可爱设计**：粉色系、圆角设计、手写风格字体

## 🚀 快速开始

### 前置要求

- Node.js 16+ 和 npm
- 现代浏览器（Chrome、Safari 等）

### 安装和启动

1. **进入项目目录**
   ```bash
   cd beauty-journey-app
   ```

2. **安装后端依赖**
   ```bash
   cd backend
   npm install
   ```

3. **初始化数据库**
   ```bash
   npm run init-db
   ```

4. **启动后端服务**
   ```bash
   npm start
   ```
   后端会在 `http://localhost:3001` 运行

5. **安装前端依赖（新开一个终端窗口）**
   ```bash
   cd frontend
   npm install
   ```

6. **启动前端开发服务器**
   ```bash
   npm run dev
   ```
   前端会在 `http://localhost:3000` 运行

## 📖 使用说明

### 用户端流程

1. **首次进入** → 欢迎页面 → 点击"加入美丽之旅"
2. **设置信息** → 输入昵称、选择开始日期、选择打卡项目、写下初始心情
3. **每日打卡** → 完成打卡项目、写下感想、上传照片
4. **查看历史** → 查看21天打卡日历，点击日期查看详情

### 默认打卡项目

- 💧 护肤：白天涂2次精华棒
- 🌙 作息：11:30前入睡
- 🏃‍♀️ 运动：每周3次 + 日步数≥8000
- 🥗 饮食：晚8点后不进食
- 💦 喝水：每日1500ml温水
- 🧘‍♀️ 体态：靠墙站立5分钟
- 🍭 戒糖：少奶茶甜食

### 管理员后台

1. 访问 `http://localhost:3000/admin`
2. 默认账号：`admin` / `admin123`
3. 功能：查看用户列表、查看打卡详情、下载CSV数据

## 🛠 技术栈

### 后端
- Express.js - Web框架
- SQLite3 - 数据库
- Multer - 文件上传
- JWT - 用户认证

### 前端
- React 18 - UI框架
- Tailwind CSS - 样式框架
- Vite - 构建工具
- React Router - 路由
- Lucide React - 图标
- Day.js - 日期处理
- html-to-image - 分享图片生成
- Canvas Confetti - 彩带效果

## 💡 微信/抖音群分享功能

### 当前实现
1. **分享图片生成**：使用 html-to-image 库将打卡页面转为图片
2. **下载保存**：用户可以保存打卡结果到手机相册
3. **群分享**：用户手动将保存的图片分享到微信群或抖音

### 进阶集成（可选）
如需更深度的微信/抖音集成，需要：

1. **微信小程序**：需要微信开发者账号和小程序资质
2. **微信公众号**：接入微信JS-SDK，实现一键分享
3. **抖音小程序**：接入抖音开放平台

## 📝 注意事项

- 首次启动前需要运行 `npm run init-db` 初始化数据库
- 用户上传的图片保存在 `backend/uploads` 目录
- 修改管理员密码请编辑初始化脚本或数据库
- 生产环境建议使用真实的数据库（PostgreSQL/MySQL）
- 生产环境需要设置合适的 JWT_SECRET
