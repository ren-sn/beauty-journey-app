## 🚀 如何运行

### 前置要求
- Node.js 16+ (建议使用18或更高版本)
- npm (通常随Node.js一起安装)

### 步骤1: 安装后端
```bash
cd backend
npm install
npm run init-db   # 初始化数据库
npm run dev      # 启动开发服务器 (端口 3001)
```

### 步骤2: 安装前端
```bash
cd frontend
npm install
npm run dev      # 启动开发服务器 (端口 5173)
```

### 步骤3: 开始使用
1. 在浏览器中打开 `http://localhost:5173`
2. 输入昵称，点击"加入美丽之旅"
3. 选择打卡项目，开始你的21天变美之旅！
